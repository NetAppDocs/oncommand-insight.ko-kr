= 데이터베이스 객체 수 계산
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
OnCommand Insight 데이터베이스의 개체 수를 확인하려면 배율 계산 기능을 사용합니다.



== 단계

. Insight 도구 모음에서 * Admin * 을 클릭하고 * Troubleshooting * 을 선택합니다.
. 다른 작업 영역의 페이지 하단에서 * 고급 문제 해결 * 링크를 클릭합니다.
. 고급 문제 해결 페이지에서 * 보고서 * 탭을 클릭합니다.
. 계산된 배율 * 을 클릭합니다.
