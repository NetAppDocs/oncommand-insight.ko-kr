= Insight syslog 콘텐츠
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
서버에서 syslog를 활성화하여 활용률 및 트래픽 데이터를 포함한 Insight 위반 및 성능 경고 메시지를 수집할 수 있습니다.



== 메시지 유형

Insight syslog에는 세 가지 유형의 메시지가 나열됩니다.

* SAN 경로 위반
* 일반 위반
* 성능 경고




== 데이터가 제공됩니다

위반 설명에는 관련 요소, 이벤트 시간, 위반의 상대적 심각도 또는 우선 순위가 포함됩니다.

성능 알림에는 다음 데이터가 포함됩니다.

* 활용률
* 트래픽 유형
* 트래픽 속도(MB
