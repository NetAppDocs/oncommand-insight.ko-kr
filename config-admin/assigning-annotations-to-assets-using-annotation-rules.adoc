= 주석 규칙을 사용하여 자산에 주석 지정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자가 정의한 기준에 따라 자산에 주석을 자동으로 할당하려면 주석 규칙을 구성합니다. OnCommand Insight는 이러한 규칙에 따라 자산에 주석을 할당합니다. 또한 Insight에서는 두 가지 기본 주석 규칙을 제공합니다. 이 규칙은 필요에 맞게 수정하거나 사용하지 않으려는 경우 제거할 수 있습니다.
