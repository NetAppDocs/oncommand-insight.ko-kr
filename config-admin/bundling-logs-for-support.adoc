= 지원을 위한 로그 번들링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Insight 소프트웨어의 문제를 해결할 때 "gz" 형식을 사용하여 NetApp 고객 지원 본부에 전송되는 로그 및 수집 기록의 zip 파일을 신속하게 생성할 수 있습니다.



== 단계

. Insight 도구 모음에서 * Admin * 을 클릭합니다.
. 문제 해결 * 을 클릭합니다.
. 데이터 보내기/수집 섹션에서 * 번들 로그 * 를 클릭합니다.
. 파일 저장 * 을 클릭합니다.
. 확인 * 을 클릭합니다.
