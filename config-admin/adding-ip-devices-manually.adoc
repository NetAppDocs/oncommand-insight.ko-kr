= 수동으로 IP 장치 추가
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
IP 식별 화면에서 사용할 수 있는 수동 추가 기능을 사용하여 IP 장치를 OnCommand Insight에 수동으로 추가할 수 있습니다.



== 단계

. Insight 웹 UI에 로그인합니다.
. Manage * > * Device resolution * 을 클릭합니다
. IP 식별 * 탭을 클릭합니다.
. 추가 버튼을 클릭합니다.
+
장치 추가 대화 상자가 표시됩니다

. 주소, IP 주소 및 고유한 장치 이름을 입력합니다.




== 결과

입력한 장치가 IP 식별 탭의 장치 목록에 추가됩니다.
