= 공급업체별 데이터 소스 참조
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
구성 세부 정보는 추가되는 데이터 소스의 공급업체 및 모델에 따라 달라집니다.

공급업체의 데이터 원본에 특별한 요구 사항 및 특정 명령과 같은 고급 Insight 구성 지침이 필요한 경우 이 정보가 이 섹션에 포함됩니다.
