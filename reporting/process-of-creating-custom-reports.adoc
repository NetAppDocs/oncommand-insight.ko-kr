= 보고서 작성 프로세스
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
임시 보고서를 만드는 프로세스에는 여러 작업이 포함됩니다.

* 보고서 결과를 계획합니다.
* 결과를 지원하는 데이터를 식별합니다.
* 데이터가 포함된 데이터 모델(예: 차지백 데이터 모델, 재고 데이터 모델 등)을 선택합니다.
* 보고서의 데이터 요소를 선택합니다.
* 보고서 결과의 서식 지정, 정렬 및 필터링 옵션을 선택할 수 있습니다.
