= 스토리지 풀을 검토합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
스토리지 풀 랜딩 페이지를 살펴보면 상호 연결된 자산에서 볼 수 있는 것과 동일한 IOPS 급증이 나타납니다. 리소스 섹션에서 이 스토리지 풀 랜딩 페이지가 여행 애플리케이션의 볼륨에 링크되어 있음을 확인할 수 있습니다. 볼륨을 클릭하여 볼륨 랜딩 페이지를 엽니다.
